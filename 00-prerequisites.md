# Pre-requisites for SparsePainter

See the main [Readme](README.md) for full context.

## Environment

On Windows, you will need to set up a working command line infrastructure. [Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install) is one solution, as is [Git Bash](https://git-scm.com/install/windows). Either can be effective.

On Mac or Linux, you already have this - just run your favorite Terminal.

You will also need a working buildchain, i.e. C++, make, etc.

1. Install [SparsePainter](https://github.com/yaolingYang/sparsePainter)
2. [Finestructure v4](https://github.com/danjlawson/finestructure4),
3. [pbwt](https://github.com/richarddurbin/pbwt).
