# A tutorial for SparsePainter

[SparsePainter](https://github.com/YaolingYang/SparsePainter) is a powerful tool for local ancestry inference at scale, described in the paper [ Yang, Y., Durbin, R., Iversen, A.K.N & Lawson, D.J. Sparse haplotype-based fine-scale local ancestry inference at scale reveals recent selection on immune responses. Nature Communications 16, 2742 (2025).](https://www.nature.com/articles/s41467-025-57601-3)

The [code](code) directory contains some useful scripts that make working with SparsePainter easier.

The [simple_europe_example](simple_europe_example) directory contains a full working example designed to show how to work with SparsePainter effectively. You should start there!
